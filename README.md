# 🪙 Finance Manager

A finances manager with a dashboard and loan payment simulation.

## Usage

### Clone the repository

- `git clone https://github.com/cwrightlibrary/FinanceManager.git`
- `cd FinanceManager`

### Create a virtual environment

#### Windows

- `py -m venv .venv`
- `.venv\Scripts\activate`

#### macOS

- `python3 -m venv .venv`
- `source .venv/bin/active`

### Install dependencies

- `pip3 install --upgrade pip`
- `pip3 install -r requirements.txt`

### Run the `streamlit` GUI

- `streamlit run app.py`
- Visit `http://localhost:8501` in your browser

From here, setup your profile and add bank accounts, income, loans, and bills. You'll have a dashboard view of your finances and a *Debt Wizard* simulator to help you plan off your loans.
